Inception of the AI Jurist

May 2023: The idea of the AI jurist app is conceived, aiming to provide legal assistance using AI technology.

Assembling the Datasets

June 2023: Around 10,000 court case datasets are collected into a CSV file as the initial dataset.

Design Creation Sprint

July witnessed an intense 10-day design phase, crafting an intuitive app interface.

Features included case outcome prediction and retrieval of similar cases, catering to user needs..

Efficient Data Management

To streamline data storage, we implemented a seamless login and register page, enhancing the app's backend capabilities.

Streamlining user experience, our app's efficient login and register page enables seamless data storage, ensuring user data privacy and accessibility.

The Quest for the Optimal Model

Experimenting with various methods like SVG and k-nearest neighbors for case prediction, we finally settled on deep neural networks with a complex architecture comprising 36 layers, including input layers, normalization layers, and dropout layers.

Initial Accuracy Challenge

The initial model accuracy was only 43%, prompting us to explore hyperparameter tuning and drop layers to optimize performance.

Leveraging Synthetic Data

Using gretel.ai, we introduced synthetic data into our dataset, adding about 60,000 accurate cases, which significantly improved accuracy by 75%

Final Model Accuracy

After further model tuning, we achieved an impressive accuracy of 74% using approximately 100,000 cases.

Exploring Web Scraping

Initially, we considered web scraping to collect data for similar cases. However, due to resource constraints, we decided against it. We opted for an alternative approach, downloading official case judgments as JSON files (50,000 cases), enabling the creation of a robust search engine.

Cosine Similarity Method

The search engine leverages the cosine similarity method to retrieve and present similar cases, enhancing user research capabilities.

Summarizing Similar Cases

To provide users with summarized insights, we fine-tuned a BERT model to generate concise summaries of the extracted similar cases.

Project Completion

The app was completed on August 2023, achieving our vision to provide AI-driven legal insights and data-driven efficiency.
